Conclusion

Where we have been and where you could go next

We have come a long way since we listed the contents of a directory in . Saving files in version control, making sure code meets style rules, running tests and managing packages… Programmers do these things every day, and we hope that understand how they work will help you do them better.

We hope your journey won't stop here. In particular, contributions of all kinds are welcome, from errata and minor improvements to entirely new sections and chapters. Please file an issue submit a pull request in our GitHub repository. Everyone whose work is incorporated will be acknowledged. Please see the contributors' guide for more information, and please note that all contributors are required to abide by our Code of Conduct.
